Aggregate Operators

  • Aggregate functions in DBMS take multiple rows from the table and return a value according to the query.
  • All the aggregate functions are used in Select statement.
  • SUM Function-This function sums up the values in the column supplied as a parameter.
  • AVG Function-This function returns the average value of the numeric column that is supplied as a parameter.
  • COUNT Function-The count function returns the number of rows in the result. It does not count the null values.
  • MAX Function-The MAX function is used to find maximum value in the column that is supplied as a parameter. It can be used on any type of data.
  • VARIANCE Function-The VARIANCE Function is used to find variance of the column specified as argument.
